# PHP based web frameworks

A set of GraphQL server implementation on popular PHP web frameworks.

## Frameworks
* Laravel

```bash
$ brew update
$ brew install homebrew/php/php70 --with-apache --with-mysql --with-postgresql --with-intl
$ php -v
```
